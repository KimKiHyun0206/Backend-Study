# MacOS
> 애플이 개발한 유닉스 기반 운영체제

* 유전적 유닉스에 속한다(최초의 유닉스의 소스 코드를 이어 받았다)
* 법률적으로도 완벽하게 유닉스 기준을 충족한다
* 현존하는 유일한 PC용 유닉스 운영체제이자 완전한 UNIX 이기 때문에 Unix-like 라고 표현하지 않는다

# MacOS : 개요
* Apple 의 Macintosh 컴퓨터에서 실행되도록 특별히 설계되었다
* 독특하고 직관적인 사용자 인터페이스를 제공한다

# MacOS : 정의
> 사용 편의성에 중점을 둔 깨끗하고 가벼운 디자인의 사용자 인터페이스
* 다양한 응용 프로그램
* Finder 를 통한 파일과 폴더 검색
* 화면 하단의 토크로 자주 사용하는 애플리케이션에 빠르게 접근

<br>

# MacOS : 내장 보안 및 데이터 보호
## 내장 보안 기능
### 게이트 키퍼
> 사용자가 유해 소프트웨어를 Mac 에 설치하지 못하도록 설계된 보안 기능

게이트 키퍼는 응용 프로그램을 실행 허용하기 전에 전자 서명을 확인함으로써 신뢰할 수 있는 소프트웨어만 시스템에서 실행할 수 있도록 한다

### FileVault
> Mac 하드디스크 드라이브 내에 있는 모든 데이터를 암호화하는 내장 암호화 기능

모든 데이터를 암호화 하면 Mac 을 분실하거나 도난당한 경우에도 데이터를 보호할 수 있다

### 방화벽
> MacOS 에는 무단으로 접근하는데 있어서 Mac 을 보호하는 방화벽이 내장되어 있다

들어오는 연결을 차단하도록 방화벽을 구성하여 해커가 시스템에 접근하는데 있어서 어렵게 만들 수 있다

### 개인 정보 기본 설정
> MacOS 에는 사용자가 자신의 데이터에 접근할 수 있는 애플리케이션을 제어할 수 있는 개인 정보 기본 설정도 있다

사용자는 연락처, 위치, 사진 및 기타 데이터에 대한 접근을 허용하거나 거부하도록 선택할 수 있다

### Safari 보안 기능
> MacOS 의 기본 웹 브라우저인 Safari 에는 몇 가지 보안 기능이 내장되어 있다
* 팝업 차단
* 사기성 웹사이트에 대해 사용자에 경고
* 교차 사이트 추적 방지 기능

<br>

## 데이터 보호
### 시스템 업데이트
> Apple 은 MacOS 용 보안 업데이트를 정기적으로 출시함으로써 취약성과 보안 위협을 해결하고 있다

좋은 업데이트이니 주기적으로 하는 것이 좋다

### 강력한 암호
> 모든 계정에 강력하고 고유한 암호를 사용하고 암호 관리자를 사용하여 암호가 노출되는 것을 막는다

### 2단계 인증
> 사용자가 암호 외에 추가적인 코드를 입렵하도록 함으로써 추가 보안 계층을 제공한다

### 소프트웨어 다운로드
> 신뢰할 수 있는 소프트웨어만 다운로드한다

### 데이터 백업
> 데이터를 외장 하드 드라이브나 클라우드 스토리지 서비스에 정기적으로 백업한다

이렇게 하면 하드웨어 오류나 보안 위반 시에도 데이터를 잃지 않을 수 있다

# MacOS : 장점
## 사용자 친화적인 인터페이스
> MacOS 는 직관적이고 사용자 친화적인 인터페이스로 알려져 있어 사용자가 필요한 정보를 쉽게 검색할 수 있다

이 시스템은 단순하고 직관적으로 설계되어 컴퓨터를 처음 접하는 사람들에게도 유용하다

## Apple 에코 시스템과의 원활한 통합
> MacOS 는 Apple 의 하드웨어 및 소프트웨어 에코 시스템과 긴밀하게 통합되어 사용자가 Apple 의 장치 및 서비스에 쉽게 접근할 수 있다

Handoff, Continuity 및 AirDrop 과 같은 기능이 포함되어 있어 사용자가 여러 장치 간에 파일 이동을 원활하게 할 수 있습니다.

## 높은 보안성
> MacOS는 폐쇄형 소스 특성과 탑재된 보안 기능으로 인해 다른 운영 체제보다 더 안전한 것으로 유명하다

사용자가 잠재적인 유해 소프트웨어를 설치하지 못하도록 방지하는 Gatekeeper 와 사용자 데이터를 암호화하여 도난이나 무단 접근으로부터 보호하는 FileVault 가 포함됩니다.

## 강력한 내장 애플리케이션
> MacOS 에는 Safari, Mail, 사진 등을 포함한 강력한 내장 애플리케이션 세트가 함께 제공 된다

이러한 애플리케이션은 MacOS 환경에 최적화되어 있습니다.

## 고품질 하드웨어
> MacOS는 높은 품질과 안정성으로 유명한 Apple 하드웨어에서만 실행될 수 있도록 설계되었다

사용자가 Mac 컴퓨터에서 MacOS를 사용할 때 원활하고 안정적인 성능을 경험할 수 있음을 의미합니다.

# MacOS : 단점

## 제한된 호환성
> MacOS는 Apple 장치에서만 호환되므로 다른 브랜드의 컴퓨터나 장치를 선호하는 사용자에게는 단점이 될 수 있다

윈도우 또는 기타 운영 체제에서만 실행되도록 설계된 특정 소프트웨어 응용 프로그램 또는 게임의 가용성을 제한할 수도 있습니다.


## 제한된 사용자 지정
> MacOS는 일부 사용자 지정 옵션을 제공하지만, 일반적으로 다른 운영 체제보다 사용자 지정 가능성이 작다

컴퓨팅 경험을 개인화하려는 사용자에게는 단점이 될 수 있습니다.

## 높은 가격
> Apple 의 하드웨어 및 소프트웨어는 다른 제조업체의 장치나 소프트웨어보다 더 비싼 경향이 있다

## 제한된 게임 옵션
> MacOS 는 다양한 게임을 지원하지만, 일반적으로 게임 친화적인 운영 체제로 간주되지 않다

대부분 윈도우에 최적화되어 있기 때문에 MacOS 에서 실행되도록 설계된 게임의 가용성이 제한적이다

## 학습 곡션
> MacOS는 사용자 친화적인 운영체제이지만 다른 운영 체제를 사용하던 사용자가 MacOS를 사용하기 위해 익숙해지기 위해서는 시간이 걸릴 수 있다

이 학습 곡선은 친숙한 컴퓨팅 환경을 선호하는 사용자에게는 단점이 될 수 있습니다.
